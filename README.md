## Features

- Easy to set up your groups or classes
- Cross-platform
- Lightweight
- Simple and Fast
- Import and Export Database in JSON file
- import Students from xlsx file
- Export Attendance by Session or Total
- Record Attendance with Manually or with QRcode Scanner
